# file-to-base64-cli

<b>سكربت يقوم بتحويل جميع الملفات الى base64 ويقوم بإضافتها في ملف json </b><br>
<br><br>




## طريقة الإستعمال 

``` bash

> npm i -g file-to-base64-cli

// Go to the folder containing the files

> file-to-base64-cli

```

<br><br>

<b>مثال لملف base64.json وملف FileArray.json</b>
